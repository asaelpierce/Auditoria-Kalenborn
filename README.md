# Portal SGA — Kalenborn do Brasil

Sistema de Gestão de Auditoria Interna (ISO 9001:2015)  
Unidade Vespasiano — React + Vite + Tailwind + Supabase

## Stack

- **React 18** + **Vite 6** (build)
- **Tailwind CSS 3** (estilização)
- **lucide-react** (ícones)
- **SheetJS / xlsx** (exportação Excel)
- **Supabase** (banco de dados + storage)

## Subir localmente

```bash
npm install
npm run dev
```

Acesse: http://localhost:5173

## Deploy na Vercel

1. Faça push deste repositório para o GitHub
2. Importe o projeto em https://vercel.com/new
3. A Vercel detecta o Vite automaticamente — clique **Deploy**

O arquivo `vercel.json` já está configurado.
